# Description du projet — Tétouan Créative

Tétouan Créative est un blog géré par une administration qui publie et met en avant les articles d’artistes et d’artisans de Tétouan.
Les visiteurs peuvent découvrir les créations, aimer, commenter et ajouter des articles à leurs favoris.
L’objectif est de promouvoir les talents locaux à travers une galerie numérique simple et interactive.
